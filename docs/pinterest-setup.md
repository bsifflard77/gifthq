# GiftHQ Pinterest Marketing Setup Guide

**Goal:** Build organic traffic through gift idea content + drive waitlist signups

---

## Step 1: Create Pinterest Business Account

1. Go to https://business.pinterest.com
2. Create new account or convert existing personal account
3. Business name: **GiftHQ**
4. Website: **gifthq.ai**
5. Select category: **App** or **Shopping**

---

## Step 2: Claim Website

1. Pinterest Settings → Claimed accounts
2. Add gifthq.ai
3. Add meta tag or upload HTML file to verify
4. This enables rich pins and analytics

## Step 5: Pin Strategy

### Pin Types

**1. Product Pins (60%)**
- Curated gift ideas from Amazon, Target, etc.
- Use affiliate links (when approved)
- Include price in description

**2. Blog/Content Pins (25%)**
- "10 Best Gifts for Coffee Lovers"
- "How to Plan Christmas Without Stress"
- Link to gifthq.ai blog (create content)

**3. App Promo Pins (15%)**
- GiftHQ features and screenshots
- Waitlist signup CTAs
- Testimonials (when available)

### Pin Design Best Practices

- **Size:** 1000 x 1500px (2:3 ratio)
- **Text overlay:** Bold, readable headline
- **Branding:** Small GiftHQ logo in corner
- **Colors:** Consistent with brand (emerald/teal)

### Pin Templates (Canva)

Create templates for:
1. Product feature pin (photo + price + "Shop now")
2. Gift guide pin (collage + "10 Best Gifts for...")
3. Tip pin (icon + short tip text)
4. App screenshot pin (phone mockup)

---

## Step 6: Content Calendar (First Month)

### Week 1: Foundation
- Pin 10 pins to each core board (curated content)
- Create 5 original "gift idea" pins
- Total: 50+ pins

### Week 2: Growth
- Pin 5 new pins daily
- Start "Gifts for Mom" content push
- Create first blog post → pin it

### Week 3: Engagement
- Pin 5 new pins daily
- Engage with similar accounts
- Join group boards (if available)

### Week 4: Optimization
- Review Pinterest analytics
- Double down on high-performing content
- Create more of what works

## Step 8: Affiliate Integration

### When Amazon Associates Approved:

1. Create pins with affiliate links
2. Use Amazon product images (with attribution)
3. Disclose affiliate relationship in pin description
4. Example: "As an Amazon Associate, I earn from qualifying purchases"

### Pin Description Template:
```
🎁 [Product Name] - The perfect gift for [recipient]!

✨ [Key feature 1]
✨ [Key feature 2]
✨ [Key feature 3]

💰 $[Price] on Amazon

#GiftIdeas #[Occasion]Gifts #GiftHQ

[Affiliate link]
```
