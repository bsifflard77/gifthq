## 🎯 GoWish — Primary Competitor

### Overview
- **Users:** 13.6M+ registered (6.2M in US alone)
- **Founded:** 2015 (as Ønskeskyen by PostNord Denmark)
- **HQ:** Copenhagen, Denmark
- **Team:** 90 employees
- **Funding:** Capital D (PE) owns 1/3, Dotcom Capital owns rest
- **Profitability:** $1.7M net profit FY2024

### Mission
"Fixing gifting" — make users' dreams come true while eliminating double-gifting and returns.

### Key Features
| Feature | Details |
|---------|---------|
| **Wishlist Creation** | Multiple lists for self, family, friends, occasions |
| **Add Items** | Paste URL, search products, browse inspiration feeds |
| **Reservation System** | Friends reserve items to prevent duplicate gifts |
| **Affiliate Model** | 65,000+ affiliate partnerships, 700+ brand partnerships |
| **Browser Extension** | Chrome extension for easy adding |
| **"GoWish" Buttons** | Brands adding these to their websites |
| **Social Features** | Follow others, public/private lists, share wishlists |
| **"Most Wanted"** | Highlight priority items |
| **Gift Generator** | AI-style suggestions based on age/gender/interests |

### What's Working for Them
1. **Massive scale** — 13.6M users, #2 on US App Store
2. **Affiliate revenue** — profitable without charging users
3. **Snap/TikTok/Meta marketing** — excellent paid acquisition
4. **Chrome extension** — frictionless adding
5. **Social shopping** — trend prediction aspirations

### What Users Complain About
- Inspiration feed can feel narrow (working on fixing)
- Some product categories underserved (hobbies)

## 🔑 Features GiftHQ Should Consider

### Must-Have (Table Stakes)
1. **Universal wishlist** — Add from ANY website (not locked to one store)
2. **Gift reservation** — Prevent duplicate gifts
3. **Easy sharing** — Links, QR codes, no account required to view
4. **Multiple lists** — Birthday, Christmas, Wedding, Baby Shower, etc.
5. **Mobile-first** — PWA or native app

### Differentiators (Our Edge)
1. **AI-Powered Gift Suggestions** — GoWish is just starting this. We can lead.
2. **Occasion Reminders** — Track birthdays/anniversaries, remind users to create lists AND remind gift-givers
3. **Price Drop Alerts** — No competitor does this well
4. **Affiliate Revenue Transparency** — Show users we make money on affiliate links (trust)
5. **Family Hub** — Child accounts, family gift coordination

### Nice-to-Have
- Browser extension
- Secret Santa generator
- Thank-you message automation
- Charity/donation option instead of gifts

## 💰 Business Model Analysis

### GoWish Model
- **Primary revenue:** Affiliate commissions (65K+ partnerships)
- **Secondary:** Brand partnerships ("GoWish" buttons on websites)
- **User cost:** FREE
- **Profitability:** Yes ($1.7M net FY24)

### GiftHQ Options

| Model | Pros | Cons |
|-------|------|------|
| **Affiliate-only (like GoWish)** | Free for users, proven model | Need massive scale |
| **Freemium** | Users pay for premium features | Lower adoption |
| **B2B (Shopify plugin)** | Recurring revenue from stores | Different market |
| **Hybrid** | Multiple revenue streams | More complexity |

**Recommendation:** Start with affiliate model (like GoWish) but add premium tier for:
- Price tracking alerts
- Advanced AI recommendations  
- Ad-free experience
- Priority support
